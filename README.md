# Superstore-Sales-Dashboard
**1. Project Title / Headline**
📊 Super Store Sales & Profit Analysis Dashboard
A dynamic, interactive Power BI dashboard built to analyze sales, profit, and operational performance across regions, categories, and customer segments, with a focus on trends, shipping efficiency, and product-level insights.

**2. Short Description / Purpose**
The Super Store Sales Dashboard helps you track sales performance, profit trends, and customer behavior across different regions and categories. You use it to identify top-performing segments, optimize shipping methods, and monitor monthly growth patterns.

**3. Tech Stack**

The dashboard was built using the following tools and technologies:

**• 📊 Power BI Desktop –** Main platform for creating interactive reports and visuals

**• 📂 Power Query –** Data cleaning and transformation for structured analysis

**• 🧠 DAX –** Measures for sales, profit, orders, and KPI calculations

**• 📝 Data Modeling –** Relationships between orders, customers, and product tables

**• 📁 File Format –** .pbix for development and .png for dashboard previews

**4. Data Source**
Source: Superstore Sales Dataset
The dataset includes order-level data covering sales, profit, shipping mode, region, category, sub-category, customer segment, and order dates. It enables analysis of performance across time, geography, and product hierarchy.

**5. Features / Highlights**
**• Business Problem**
Retail businesses handle large volumes of sales data across regions, products, and shipping methods. Without a structured dashboard, it is hard to track performance trends, identify profit drivers, and optimize operations.

Key questions include:

Which regions generate the highest sales and profit?

Which shipping mode is most used and efficient?

Which product categories and sub-categories drive revenue?

How do sales and profit change over time?

**• Goal of the Dashboard**

To provide an interactive analytical tool that:

Enables monitoring of sales, profit, and order trends

Supports decision-making for product and regional strategy

Identifies high and low-performing categories and segments

Improves operational insights such as shipping efficiency

**• Business Impact & Insights**

**Sales Optimization:** Focus on high-performing regions like West to maximize revenue.

**Product Strategy:** Prioritize top categories like Office Supplies and high-selling sub-categories like Phones.

**Operational Efficiency:** Standard shipping drives most sales, indicating customer preference.

**Customer Targeting:** Consumer segment contributes the highest share, useful for marketing focus.

**Trend Analysis:** Monthly trends help plan inventory and promotional strategies.
